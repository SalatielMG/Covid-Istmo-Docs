## Obtener todas la preguntas

```sh
method = get

endPoint = /question

headers = {
    token: string
}

response = {
    "200": {
        success: true | false,
        results: [
            {
                id: number,
                question: string,
                typeQuestion: number,
                isSymptom: boolean,
                value: number,
                question_id: object<Pregunta>,
            }
            ...
        ]
    }
    "401": {
        msj: 'Solicitud no autorizado'
    },
}

```
